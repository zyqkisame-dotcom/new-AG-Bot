# 🥇 AnyGold Bot

Auto WhatsApp blast harga emas kepada dealers setiap 10:00 AM MYT (Isnin–Jumaat).

## Fungsi

- Scrape harga emas terkini dari 3 sumber (AnyGold, MKS, MS Gold)
- Kira harga untuk setiap dealer berdasarkan potongan & rounding
- Blast mesej WhatsApp automatik ke 10 dealers
- Refresh harga setiap 30 minit

## Setup

### Keperluan
- Node.js v18+
- Google Chrome (installed)
- WhatsApp account

### Install

```bash
npm install
node index.js
